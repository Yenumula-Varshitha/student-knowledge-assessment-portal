# Student Knowledge Assessment Portal

## 📌 Project Overview
The Student Knowledge Assessment Portal is a web-based application developed to help students assess their knowledge through quizzes and topic-wise tests. The portal provides an interactive learning environment where users can register, log in, select topics, attempt quizzes, and view their results instantly.

This project is designed to improve learning efficiency and help students evaluate their understanding in different technical subjects.

---

## 🚀 Features
- User Registration and Login System
- Topic-wise Quiz Selection
- Multiple Choice Questions
- Instant Result Generation
- Leaderboard System
- Interactive and Responsive UI
- XML-based Question Storage
- Simple and User-Friendly Interface
